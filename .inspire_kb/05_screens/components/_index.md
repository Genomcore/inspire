# Components

The **shared component catalog** — the reusable UI pieces screens reference (link,
don't redescribe) and the prototype implements. Each entry documents the
component's purpose, API/props, and state (`implemented` / `to-extract`).

Add a component when it appears in **≥2 screens** (≥3 if trivial), via
`/inspire_screens extract component {name}`. Keep it here, not inlined in screens.
